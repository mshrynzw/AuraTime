# API仕様：休暇

## 1. 休暇申請

### POST `/api/v1/leave-requests`
休暇申請を提出。

- **権限**: Employee
- **リクエスト**:
    ```json
    {
      "leave_type_id": "uuid",
      "start_date": "2026-01-05",
      "end_date": "2026-01-06",
      "days": 2.0,
      "reason": "私用のため"
    }
    ```
- **処理**:
    - `requires_approval` が真の休暇タイプの場合、ステータスを `submitted` とし、承認リクエストを自動生成。

## 2. 休暇実績参照

### GET `/api/v1/leave-requests`
申請履歴と現在のステータスを一覧表示。

- **レスポンスに含まれるステータス**:
    - `draft`, `submitted`, `approved`, `rejected`, `canceled`

### GET `/api/v1/leave-types`
利用可能な休暇タイプ（有給、慶弔等）を取得。
